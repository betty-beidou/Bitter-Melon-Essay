# Bitter-Melon-Essay

个人写作与资料整理仓库。

## 项目内容
- `docs/`：正式文章与图表
- `drafts/`：草稿与碎片想法
- `images/`：插图或示意图（推荐用 draw.io/diagrams.net）
- `references/`：参考文献、链接与笔记

## 使用方式
- 网页端直接在对应文件夹 **Add file → Create new file** 新建/编辑 Markdown。
- 需要画图：在 `docs/` 或 `images/` 下新建 `*.drawio`，用 **Open with draw.io** 在线编辑并提交。
